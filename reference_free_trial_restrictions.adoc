= 無償トライアルの制限
:toc: macro
:hardbreaks:
:toclevels: 1
:toc: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/

[role="lead"]
無料トライアルアカウントには、次の制限事項が適用されます。

* 最大 1 つの本番インスタンス。
* 最大 1 つのサンドボックスインスタンス。
* 1 日に最大 3 つの即時バックアップを作成できます。
* 最初の 30 日間だけバックアップを自動化。
* 無償トライアル終了後の 15 日間の猶予期間中は、スケジュールされたバックアップは実行できません。
